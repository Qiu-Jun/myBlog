## 版本切换
```bash
# mac (window没有n模块，window安装nvm)
# 全局安装n
$ npm install -g n
# 升级到最新稳定版
$ n stable 
# 升级到最新版
$ n latest
# 升级到定制版
$ n v14.3.0
# 切换使用版本
$ n 14.3.0
```
