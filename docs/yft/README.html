yft-design是一款美观且功能强大的在线设计工具，具备海报设计和图片编辑功能，基于fabric.js v6的开源版【稿定设计】。适用于多种场景，如海报生成、电商产品图制作、文章长图设计、视频/公众号封面编辑等.适配稿定设计导出pdf还原、支持导入psd、导出图片，svg，pdf

目前我们项目还提供psd解析，pdf解析和ai抠图等开放免费接口. 大家感兴趣的话可以一起维护这款小编辑器，欢迎star ​
